# 🚀 **Мультифункциональное Приложение: Управление финансами и Тугрики**

## ✨ Обзор Проекта

Это инновационное десктопное приложение, разработанное для того, чтобы помочь вам взять под контроль свои личные финансы и привнести элемент игры в повседневные задачи. Мы верим, что управление деньгами и достижение целей может быть увлекательным и мотивирующим процессом. Приложение объединяет в себе строгий учет доходов и расходов с системой вознаграждений ("Тугрики") за выполнение рутинных дел.

## 🎯 Ключевые Особенности

*   **Интуитивное Управление Финансами:** Легко отслеживайте доходы и расходы, создавайте бюджеты, анализируйте свои траты с помощью наглядных круговых диаграмм и таблиц. Поддерживается импорт и экспорт данных в формате CSV.
*   **Геймификация Задач:** Превратите рутинные дела в захватывающие квесты! Устанавливайте задачи, выполняйте их и зарабатывайте виртуальную валюту — "Тугрики".
*   **Магазин Наград:** Тратьте заработанные Тугрики на приятные бонусы в встроенном магазине (например, "Вечер кино" или "Вкусный ужин").
*   **Генератор Случайных Чисел (Кубик):** Встроенный инструмент для принятия решений с красивой анимацией и историей бросков.
*   **Наглядная Аналитика:** Получайте глубокие инсайты о вашем финансовом состоянии за различные периоды (квартал, полугодие, год) благодаря интерактивным дашбордам.
*   **Темная и Светлая Темы:** Настраиваемый интерфейс для комфортного использования в любое время суток.

## 💡 Почему это приложение?

В современном мире, где финансовая грамотность и самоорганизация играют ключевую роль, это приложение предлагает уникальное решение, объединяющее практичность финансового планирования с психологией геймификации. Это не просто трекер расходов, это ваш личный мотиватор по достижению целей.

## 🛠️ Технологии

*   **Основной язык:** Python 3
*   **Графический интерфейс:** PyQt6
*   **Обработка данных:** Pandas
*   **Хранение данных:** JSON (для задач и тугриков), CSV (для финансов)

## 📂 Структура Проекта

Ниже представлено описание основных файлов и директорий в репозитории:

*   **`main.py`** — Главный исполняемый файл приложения. Содержит всю логику работы интерфейса (PyQt6), обработку данных (Pandas), отрисовку графиков и управление внутренним состоянием (задачи, тугрики, история бросков кубика).
*   **`app_internal_data.json`** — Файл локальной базы данных. Хранит информацию о текущих и выполненных задачах, балансе "Тугриков" и истории покупок в магазине наград.
*   **`requirements.txt`** — Список зависимостей Python, необходимых для запуска проекта (PyQt6, pandas).
*   **`расходы доходы.csv`** — Пример файла с финансовыми данными (история транзакций), который используется приложением для построения таблиц и графиков.
*   **`images/`** — Директория, содержащая графические ресурсы (картинки), используемые в магазине наград (например, `Вечер кино.png`, `Вкусный ужин.png`).
*   **`Logo.png` / `logo.ico`** — Иконки приложения, отображаемые в окне программы и на панели задач.
*   **`LICENSE`** — Файл с текстом лицензии (MIT), под которой распространяется данный проект.
*   **`README.md`** — Этот файл с описанием проекта, инструкциями по установке и структурой репозитория.

## 🚀 Начало Работы

Чтобы запустить проект локально, следуйте этим шагам:

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/MrBombastick08/Graduation-thesis.git
    cd Graduation-thesis
    ```
2.  **Установите зависимости:**
    Убедитесь, что у вас установлен Python 3. Затем выполните команду:
    ```bash
    pip install -r requirements.txt
    ```
3.  **Запустите приложение:**
    ```bash
    python main.py
    ```
