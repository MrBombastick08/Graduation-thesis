# 🚀 **Мультифункциональное Приложение: Ваш Персональный Финансовый Ассистент и Геймифицированный Планировщик Жизни**

## ✨ Обзор Проекта

Это инновационное мультифункциональное приложение, разработанное для того, чтобы помочь вам взять под контроль свои личные финансы и привнести элемент игры в повседневные задачи. Мы верим, что управление деньгами и достижение целей может быть увлекательным и мотивирующим процессом.

## 🎯 Ключевые Особенности

*   **Интуитивное Управление Финансами:** Легко отслеживайте доходы и расходы, создавайте бюджеты, анализируйте свои траты и планируйте сбережения с помощью наглядных графиков и отчетов.
*   **Геймификация Задач:** Превратите рутинные дела в захватывающие квесты! Устанавливайте цели, зарабатывайте очки, открывайте достижения и соревнуйтесь с друзьями, чтобы оставаться мотивированными.
*   **Персонализированные Цели:** Определяйте свои финансовые и личные цели, а FinFlow поможет вам разработать пошаговый план их достижения.
*   **Наглядная Аналитика:** Получайте глубокие инсайты о вашем финансовом состоянии и прогрессе в задачах благодаря интерактивным дашбордам.
*   **Безопасность и Конфиденциальность:** Ваши данные защищены с использованием передовых технологий шифрования и строгих протоколов безопасности.

## 💡 Почему это приложение?

В современном мире, где финансовая грамотность и самоорганизация играют ключевую роль, это приложение предлагает уникальное решение, объединяющее практичность финансового планирования с психологией геймификации. Это не просто приложение, это ваш личный тренер по достижению целей и финансовой независимости.

## 🛠️ Технологии

*   **Основной язык:** Python
*   **Графический интерфейс:** PyQt6
*   **Обработка данных:** Pandas
*   **Хранение данных:** JSON-файлы

## 🚀 Начало Работы

Чтобы запустить проект локально, следуйте этим шагам:

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/MrBombastick08/Graduation-thesis.git
    cd Graduation-thesis
    ```
2.  **Установите зависимости:**
    ```bash
    pip install -r requirements.txt
    ```
3.  **Запустите приложение:**
    ```bash
    python main.py
    ```
